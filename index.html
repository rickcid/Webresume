MyFlix

MyFlix Application

MyFlix is an application that mimics the well known NetFlix movie on demand service. It is a production quality application featuring full payment processing, account management, and administration capabilities. It also includes service features that are popular in modern applications, including social media.

Take a look at the application's code in my GitHub repository. There you will find that I used TDD and BDD development processes.

Go ahead and click the MyFlix application image to be directed to the application's access page that includes instructions on how to log-in.

¡Publícalo!

¡Publícalo!

¡Publícalo! is a social discussion board Ruby on Rails application that mimics the well-known Reddit social news website. This application is in Spanish, and it allows users to post links to web content that other users may then vote up or down, causing the posts to appear more or less prominently on the page.

Take a look at the application's code in my GitHub repository.

Go ahead and click the application image to be directed to the ¡Publícalo! front page.

Use U: john@example.com P: letmein to access the voting functionality.
